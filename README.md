# FilmFlip - Рандомайзер фильмов 🎬

**Не знаешь что посмотреть? Доверься случаю!**

FilmFlip - это веб-приложение для случайного выбора фильмов с использованием TMDb API. Сайт полностью адаптирован для мобильных устройств и готов к размещению на GitHub Pages.

## ✨ Возможности

- 🎲 **Случайный выбор фильма** с учётом ваших предпочтений
- 🎭 **Фильтрация по жанрам** (все жанры из TMDb)
- 📅 **Выбор десятилетия** (1950-е - 2020-е)
- ⭐ **Фильтр по рейтингу** (0-10)
- 🎬 **Просмотр трейлера** на YouTube
- 📺 **Где смотреть** - информация о стриминговых платформах в России
- 📤 **Поделиться** - скопируйте рекомендацию для друзей
- 🌙 **Тёмная тема** - кинематографичный дизайн
- 📱 **Mobile-first** - отлично работает на телефонах

## 🚀 Быстрый старт

### 1. Получите API ключ TMDb (бесплатно)

1. Перейдите на [https://www.themoviedb.org/](https://www.themoviedb.org/)
2. Зарегистрируйтесь (если нет аккаунта)
3. Перейдите в настройки профиля → **API**
4. Нажмите "Create" или "Request an API Key"
5. Выберите "Developer" (для личного использования)
6. Заполните форму (можно кратко описать проект)
7. После одобрения скопируйте **API Key (v3 auth)**

> ⏱ Обычно ключ выдаётся мгновенно или в течение нескольких минут.

### 2. Размещение на GitHub Pages

#### Вариант A: Через GitHub UI

1. Создайте новый репозиторий на GitHub
2. Загрузите файлы проекта (`index.html`, `style.css`, `script.js`, `README.md`)
3. Перейдите в **Settings** → **Pages**
4. В разделе "Source" выберите ветку `main` и папку `/ (root)`
5. Нажмите **Save**
6. Через 1-2 минуты сайт будет доступен по адресу: `https://ваш-username.github.io/имя-репозитория/`

#### Вариант B: Через Git командную строку

```bash
# Инициализация репозитория (если ещё не инициализирован)
git init
git add .
git commit -m "Initial commit"

# Добавьте удалённый репозиторий
git remote add origin https://github.com/ВАШ_USERNAME/ВАШ РЕПОЗИТОРИЙ.git

# Отправьте код
git push -u origin main
```

Затем включите GitHub Pages в настройках репозитория.

### 3. Использование

1. Откройте сайт в браузере
2. Введите ваш API ключ от TMDb (сохраняется в localStorage)
3. Настройте фильтры (жанр, десятилетие, рейтинг)
4. Нажмите **"🎲 Найти фильм"** или **"⚡ Мне повезёт!"**
5. Смотрите информацию о фильме, трейлер и где посмотреть!
